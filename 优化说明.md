## 🚀 如何使用优化后的文档

### 启动本地服务器
```bash
# 在项目根目录运行
npx docsify serve

# 或使用全局安装的 docsify
docsify serve
```

### 访问文档
打开浏览器访问：`http://localhost:3000`

### 主要功能
- 🔍 **搜索**：顶部搜索框，支持全文检索
- 📂 **导航**：左侧侧边栏，按分类浏览
- 📄 **翻页**：页面底部，上一页/下一页
- 📋 **复制**：代码块右上角，一键复制
- 🖼️ **缩放**：点击图片放大查看
